# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
Play with your friend and enjoy the classic game with a clean UI.

---

## 🚀 Features

- 🟢 Two-player mode (X vs O)
- 🔄 Reset game option
- 🆕 New game button
- 🎉 Winner announcement popup
- 🤝 Draw detection
- 💻 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
